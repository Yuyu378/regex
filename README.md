# ISO8601 [calender dates](https://en.wikipedia.org/wiki/ISO_8601#Calendar_dates)
```regex
(?P<extended_calendar_dates>(?:\d{2}(?:0[1235679]|[2468][1235679]|[13579][0134578])|(?:0[1235679]|[2468][1235679]|[13579][0134578])00)-(?:(?:11|0[469])(?:-(?:30|[12]\d|0[1-9]))?|(?:1[02]|0[13578])(?:-(?:3[01]|[12]\d|0[1-9]))?|02(?:-(?:2[0-8]|1\d|0[1-9]))?)|(?:\d{2}(?:0[48]|[2468][048]|[13579][26])|(?:0[48]|[2468][048]|[13579][26])00)-(?:(?:11|0[469])(?:-(?:30|[12]\d|0[1-9]))?|(?:1[02]|0[13578])(?:-(?:3[01]|[12]\d|0[1-9]))?|02(?:-(?:[12]\d|0[1-9]))?))
```
Accept format `YYYY-MM-DD`, `YYYY-MM`
